## Flexure & Isostasy Modeler ##
This project is to stimulate as a scientific modeling tool for lithospheric flexure and isostatic response under a surface load. By utilizing Python, NumPy, Matplotlib, SQLite in order to calculate, plot, and save within a database, lithospheric deflection results, and topography of the load.

## Project Goal ##
The overall goal from this project was to model how the lithosphere bends under different loads like a mountain, volcano, ice sheet, or ETC.. is placed on top of it.

This project allows the person to stimulate with different values:
- Elastic Thickness
- Load Height
- Load Width

In addition to having comparison models play for lithospheric deflection where the user can manipulate the following values:
- Elastic Thickness for Stimulation 1
- Elastic Thickness for Stimulation 2
- Elastic Thickness for Stimulation 3

## Scientific Background ##
Lithospheric flexure describes how Earth's outer rigid shell bends under weight. A thicker or stronger lithosphere bends differently than a thinner or weaker lithosphere.

By using a simplifed version of a 1D flexure model, it is achievable to find and observe the changes in elastic thickness against deflection.
